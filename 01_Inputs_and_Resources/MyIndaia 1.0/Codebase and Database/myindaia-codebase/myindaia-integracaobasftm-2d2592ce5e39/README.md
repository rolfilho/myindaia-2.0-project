# README #

Arquivo config.ini. O sistema vai gravar sempre a última data de envio do arquivo, só enviará novamente de forma automática no próximo dia, no horário igual ou maior que o cadastrado;
Os arquivos json de envio e de resposta são gravados na subpasta "Arquivos";
Os arquivos das pastas Log e Arquivos serão sempre excluídos após 60 dias;
As informações de Endereço do WebService e o Token estão gravados direto no código.
